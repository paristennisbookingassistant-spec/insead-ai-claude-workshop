## Two files. That's all you need.

### 1. **[INSTALL.html](INSTALL.html)** — Install Claude Code on Sunday or Monday

Two downloads + one click in the VS Code Extensions panel. ~10 minutes. Plain English, no terminal commands.

### 2. **[WORKSHOP.html](WORKSHOP.html)** — Three master prompts for Tuesday night

Pick one. Paste it into Claude. Claude does the rest — clones this repo, installs the skill, builds your thing. The page includes a one-click Copy button for each prompt.

(Plain-markdown version: [WORKSHOP.md](WORKSHOP.md))

That's the whole workflow.
