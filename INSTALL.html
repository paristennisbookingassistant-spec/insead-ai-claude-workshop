Install Claude Code in ~10 minutes.

No prior coding experience needed. Two downloads, one click in the Extensions panel. That's it.

Before you start

You need:

  • A laptop (Mac or Windows). Both work.
  • Admin rights to install software. (Personal laptop recommended — work laptops with IT restrictions may block this.)
  • A paid Claude account. Claude Code requires Pro, Max, Team, Enterprise, or a Console (API) account. The free tier does not include Claude Code access. Claude Pro at claude.ai ($20/month) is the easiest option, and it's plenty for the workshop and the rest of your MBA.
  • About 10 minutes.

One thing worth knowing

Installing the VS Code extension automatically gives you Claude Code in two ways: the graphical chat panel inside VS Code and the `claude` command in any terminal. You don't have to choose. The extension is the easiest way to get both at once.

The happy path

Three steps. About 10 minutes. No terminal commands.

  1. 1
    Download & install VS Code from code.visualstudio.com/download. Click Next through the installer.
  2. 2
    Open VS Code → Extensions panel → search "Claude Code" → click Install. The extension is published by Anthropic.
  3. 3
    Sign in with your Claude account when prompted. A Claude icon appears in your left sidebar. Click it to start chatting.

1Install VS Code

VS Code is a free code editor made by Microsoft. It's the most popular editor for working with Claude Code because the Claude extension installs straight into it, gives you a chat panel on the side, and lets you see files Claude creates in the file browser.

What's VS Code in plain English?

Think of it like Microsoft Word, but for code instead of documents. You'll mostly use it to open the Claude chat panel and click around. You don't need to "learn" VS Code.

Install steps

  1. Go to code.visualstudio.com/download.
  2. Click the big blue button for your laptop type (Mac or Windows). An installer downloads.
  3. Open the file from your Downloads folder. Click Next through the installer.
    • Windows: when you see checkboxes, leave the defaults. Make sure "Add to PATH" stays checked.
    • Mac: drag the VS Code icon into Applications when prompted. Mac may warn it's from the internet — click Open.
  4. Launch VS Code. You see a welcome page.

Minimum version

The Claude extension needs VS Code 1.98 or higher. Anything you download from the link above this week will be fine. If you already have an older VS Code, update it through Help → Check for Updates.

✓ Check

VS Code is open. You see the welcome page with "Get Started" tabs. The window background is dark blue or grey.

2Install the Claude Code extension

This is the part that does the heavy lifting. The extension installs Claude Code, the official VS Code integration, in one click. No terminal commands.

Install steps

  1. In VS Code, look at the left sidebar — there's a column of icons. Click the four-squares icon labeled Extensions. (Or press Cmd+Shift+X on Mac, Ctrl+Shift+X on Windows.)
  2. A search box appears at the top of the Extensions panel. Type: Claude Code
  3. The first result should be Claude Code by Anthropic. Click it.
  4. Click the blue Install button.
  5. Wait ~30 seconds. The button changes to Uninstall when done. You may see a small notification popup at the bottom right.
I don't see "Claude Code by Anthropic" in the results

Make sure you're spelling it as two words with a space. There are other extensions called "Claude" or "Claude Sonnet" — the official one is specifically Claude Code and the publisher is Anthropic.

If you really can't find it, the extension page is at marketplace.visualstudio.com/items?itemName=anthropic.claude-code — open that link in your browser and click "Install" there; it'll launch VS Code and install the extension.

✓ Check

A small Claude icon (a stylized "C" or sunburst) appears in the left sidebar of VS Code, below the Extensions icon.

3Sign in to your Claude account

Click the Claude icon in the left sidebar. A panel opens on the right. It asks you to sign in.

  1. Click Sign in. A browser tab opens.
  2. Log in with your Claude account (same one you use for claude.ai).
  3. The browser confirms login. Switch back to VS Code.
  4. You see a welcome message and a chat input box.

✓ You're done

Type something into the chat box, like "what can you do?", and press Enter. Claude responds. You're ready for Tuesday.

+Optional: extras worth setting up

Create your global CLAUDE.md

Claude reads this file at the start of every session, in every project. It's where you teach Claude your preferences once so you never have to repeat yourself.

  1. Download starter-files/CLAUDE-md-starter.md from this repo.
  2. Open it. Fill in the parts in <angle brackets>.
  3. Save it as CLAUDE.md in the .claude folder in your home directory:
    • Mac: ~/.claude/CLAUDE.md
    • Windows: C:\Users\<you>\.claude\CLAUDE.md

If the .claude folder doesn't exist yet, the first time you use Claude Code creates it automatically. You can also just ask Claude in the chat panel: "Help me create my global CLAUDE.md file." — Claude will guide you through it.

I'd rather use the terminal version of Claude Code

You don't need this if you installed the VS Code extension above — the extension automatically installs the claude command too. Just open the terminal (in VS Code: Cmd/Ctrl+`) and type claude.

If you want to skip VS Code entirely and only use the terminal version, use the official native install:

Mac / Linux / WSL:

curl -fsSL https://claude.ai/install.sh | bash

Windows PowerShell:

irm https://claude.ai/install.ps1 | iex

Then type claude in any terminal. This is the official "native install" recommended by Anthropic. No Node.js needed. Auto-updates in the background.

?Stuck?

I can't find the Extensions icon in VS Code

It's on the very far left, in the activity bar. If the activity bar is hidden, press Ctrl+Shift+E (Windows) or Cmd+Shift+E (Mac) — or go to View → Open View → Extensions from the top menu.

The Claude extension installed but I don't see the icon

Restart VS Code completely (Quit and reopen, not just close the window). The Claude icon should appear in the left sidebar after the restart.

If still missing, look in the View → Command Palette menu (Cmd/Ctrl+Shift+P), type "Claude", and select Claude Code: Open in New Tab. This forces the extension to load.

Sign-in keeps failing / the browser doesn't open

Try these in order:

  1. Restart VS Code.
  2. Make sure you're logged into claude.ai in your default browser first, then try again.
  3. If you're on a corporate or school network, try a different network (your phone hotspot is a quick test). Some networks block the OAuth callback.
